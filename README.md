# LitReview

A literature review / research notes project.

## Structure

- `sources/` — saved papers, articles, PDFs, and citation info
- `notes/` — reading notes, summaries, and synthesis per topic
- `drafts/` — draft write-ups of the review

## Workflow

1. Add source material to `sources/`.
2. Take notes per source (or per topic) in `notes/`.
3. Synthesize findings into `drafts/`.
